Detailed Description

template<typename T>
class Vetor< T >

Classe template para um vetor genérico.

Template Parameters
TO tipo de dado dos elementos a serem armazenados no vetor.

O Vetor armazena até 100 elementos do tipo T.

Adiciona um elemento em uma posição específica do vetor.

Parameters
indexO índice onde o elemento será adicionado.
valorO valor a ser adicionado.
Exceptions
std::out_of_rangeSe o índice estiver fora dos limites válidos (0 a quantidade atual) ou se o vetor estiver cheio.

Se o índice for igual à quantidade atual de elementos, o elemento é adicionado ao final. Caso contrário, os elementos existentes a partir do índice são deslocados para a direita.
